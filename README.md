# TicTacToe in Java

A simple console-based TicTacToe game written in Java.  
This project demonstrates basic game logic, user input handling, and win/draw detection.

---

## Features
- Two-player mode (Player X vs Player O)
- Input validation (prevents overwriting moves)
- Win/draw detection
- Clear console board display

---

## Tech Stack
- Language: Java
- Environment: Console/Terminal

---

## How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/hasvanthreddy/tictactoe.git
   cd tictactoe
2. Compile the code
   javac tictactoe.java

3. Run The Game
   java tictactoe
